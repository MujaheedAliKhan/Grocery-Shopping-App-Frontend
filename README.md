# 🎨 Frontend - Grocery Shopping App

This is the frontend of the **Grocery E-Commerce Application** built using **React.js** and **Tailwind CSS**.

It provides a responsive shopping experience for users, along with admin product management and an AI-powered chatbot.

---

## 🚀 Features

### 👤 Authentication
- User Registration
- User Login
- Persistent Login (JWT stored in localStorage)
- Protected Routes
- Admin Route Protection

---

### 🛒 Shopping Features
- Product Listing
- Product Details Page
- Search Functionality
- Category Filtering
- Pagination
- Add to Cart
- Cart Quantity Management
- Remove from Cart
- Buy Now
- Checkout with Payment Method Selection

---

### 📦 Orders
- Place Order
- View My Orders
- Cancel Order
- Delete Order

---

### 👨‍💼 Admin Dashboard
- Add Product
- Edit Product
- Delete Product
- View All Orders
- View Total Revenue
- Mark Orders as Delivered

---

### 🤖 AI Chatbot
- Floating Chatbot Button
- Bounce Animation
- Expand/Collapse Chat Window
- Grocery Suggestions
- Healthy Food Recommendations
- Recipe Assistance

---

### 🎨 UI Features
- Fully Responsive Design
- Toast Notifications
- Loading Spinners
- Custom 404 Not Found Page

---

## 🧱 Tech Stack

- **React.js**
- **Tailwind CSS**
- **React Router DOM**
- **React Hot Toast**
- **Context API**
- **Fetch API**

---

## ⚙️ Setup

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm run dev
```

JWT token is stored in **localStorage** and used for:

- Persistent Login
- Protected Routes
- Admin Authorization

## ✨ Notes

- Cart data stored in **localStorage**
- Authentication handled with **Context API**
- Search and filtering handled on frontend
- Fully connected with backend REST APIs
- Responsive for mobile and desktop
- Includes AI-powered grocery chatbot
